# Чайка Парк — онлайн-бронирование лежаков

## Быстрый старт с Claude Code

1. Распакуй этот скелет в пустой репозиторий, инициализируй git:
   ```bash
   git init
   git add .
   git commit -m "init: project scaffold + CLAUDE.md"
   ```

2. Скопируй `.env.example` → `.env` и заполни (можно оставить платёжные ключи пустыми на первых итерациях).

3. Запусти Claude Code в папке проекта:
   ```bash
   claude
   ```

4. Дальше работай итерациями, ссылаясь на раздел "Порядок реализации" в `CLAUDE.md`. Пример первого запроса:
   > Разверни скелет Next.js-проекта (TypeScript, App Router, Tailwind), подключи Prisma к схеме из prisma/schema.prisma, накати миграции локально через docker-compose.

5. Не переходи к интеграции Т-Кассы, пока не готов и не протестирован UI бронирования на моках — так проще отлаживать.

6. Для Т-Кассы сначала используй демо-терминал (Test mode), боевой TerminalKey подключай в самом конце, перед реальным деплоем.
